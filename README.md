# 🌳 B+ Tree Based Database System
A lightweight database system implemented in **Java**, powered by a fully custom **B+ Tree** data structure.

---

## 📌 Overview

This project implements a self-balancing **B+ Tree** from scratch and uses it as the core indexing structure of a simple database system.

The B+ Tree manages all keys and indexes, providing efficient:

- 🔍 Search
- ➕ Insert
- ❌ Delete

operations with logarithmic time complexity.

---

## 🧠 Key Features

- Full implementation of **B+ Tree**
- Automatic **Self-Balancing**
- Node splitting & merging
- Index-based data access
- Optimized search performance


---

- ☕ Java
- 🌳 Custom B+ Tree Implementation
